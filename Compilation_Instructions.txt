Steps to compile and run the spatter application:
*************************************************

Step 0 : Update cmake version to 3.25 or above
		
		 Clone the latest branch(master) of tt-metal
         cd /home/user/tt-metal

step 1 : Change directoy to 
         cd /home/user/tt-metal/tt_metal/programming_examples

Step 2 : Clone the Spatter repository
		 https://github.com/srikanthcalligo/spatter_in_compute.git
		 cd spatter_in_compute
		 git checkout compute_v1
		 
		 cd /home/user/tt-metal
		 
		 ./build_metal.sh --enable-profiler --build-programming-examples
		 

Step 3 : Run the following Tests

		cd /home/user/tt-metal
		
Scatter Tests:
**************
		
		Single Core
		***********
		TT_SPATTER_BFLOAT16_VERSION_SERIAL=1 ./build_Release_tracy/tt_metal/programming_examples/spatter_in_compute/spatter_in_compute_core -k scatter -pUNIFORM:8:1 -l $((2**24))
 	
 		Multi Core
		**********
 		TT_SPATTER_BFLOAT16_VERSION_PARALLEL=1 ./build_Release_tracy/tt_metal/programming_examples/spatter_in_compute/spatter_in_compute_core -k scatter -pUNIFORM:8:1 -l $((2**24))
 
 
Gather Tests:
*************
 		
 		(Bfloat)Single Core
 		*******************
 		TT_SPATTER_BFLOAT16_VERSION_SERIAL=1 ./build_Release_tracy/tt_metal/programming_examples/spatter_in_compute/spatter_in_compute_core -k gather -pUNIFORM:8:1 -l $((2**24))
 
 		(Bfloat)Multi Core
 		******************
 		TT_SPATTER_BFLOAT16_VERSION_PARALLEL=1 ./build_Release_tracy/tt_metal/programming_examples/spatter_in_compute/spatter_in_compute_core -k gather -pUNIFORM:8:1 -l $((2**24))
 
 		(Float)Single Core
 		******************
 		TT_SPATTER_FLOAT_VERSION_SERIAL=1 ./build_Release_tracy/tt_metal/programming_examples/spatter_in_compute/spatter_in_compute_core -k gather -pUNIFORM:8:1 -l $((2**24))
 
 		(Float)Multi Core
 		*****************
 		TT_SPATTER_FLOAT_VERSION_PARALLEL=1 ./build_Release_tracy/tt_metal/programming_examples/spatter_in_compute/spatter_in_compute_core -k gather -pUNIFORM:8:1 -l $((2**24))
 
 		
 Note : To run serial version on the host, execute the command without TT_SPATTER_* flag.
